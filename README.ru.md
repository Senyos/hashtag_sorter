# Hashtag sorter bot

Этот бот сортирует теги в алфавитном порядке.

- [Запуск бота](#запуск-бота)
- [Использование](#использование)

## Запуск бота

Бот ожидает отдельный файл `config.py` со следующим содержимым:

```python
settings = {
    'token': 'ваш_токен',
    'bot': 'имя_бота (его username)',
    'prefix': 'префикс_команд',
}
```

Он импортирует эти настройки в `main.py` для связи с ботом в телеграмме.

Далее запуск происходит через запуск файла `main.py`.

## Использование

У бота есть всего лишь одна команда — `start`, у которой есть несколько синонимов: `help`, `помощь`, `пом`. Ввод этой команды вызывает меню помощи о том, как пользоваться ботом.

Сообщения без команды бот воспринимает как набор тегов. Он делит сообщение на теги по пробелу и затем сортирует их, выводя результат.

Теги можно указывать и с и без символа "#":

```sh
#tag1 #tag2 #тег3 #тег4
```

и

```sh
tag1 tag2 тег3 тег4
```

будут обработаны идентично.
